# Arch Linux ARM для Xiaomi Redmi Note 4 (Mido)

## Описание
Этот проект представляет собой порт Arch Linux ARM для устройства Xiaomi Redmi Note 4 (codename: mido). Для загрузки используется lk2nd, который запускает extlinux. Ядро взято из проекта [debian-on-mido](https://github.com/calico-cat-3333/debian-on-mido) (версия ядра 6.16.3).

## Статус
Текущий статус порта - бета. Возможны нестабильности и ошибки.

## Требования
- Устройство: Xiaomi Redmi Note 4 (mido)
- Компьютер: Windows или Linux
- Разблокированный загрузчик (bootloader)

## Установка
**ВНИМАНИЕ: Этот процесс удалит все данные с вашего телефона. Обязательно сделайте резервную копию важных файлов перед началом.**

1.  Скачайте файлы из последнего релиза.
2.  Используйте 7-Zip (или другой архиватор, поддерживающий .7z) для распаковки rootfs.
3.  Убедитесь, что ваш телефон находится в режиме fastboot (`adb reboot bootloader`).
4.  Откройте терминал/командную строку на компьютере и выполните следующие команды:
    ```bash
    fastboot erase boot
    fastboot flash boot <файл lk2nd>
    fastboot erase system
    fastboot flash system <файл bootfs>
    fastboot erase userdata
    fastboot flash userdata <файл rootfs>
    ```
    Замените `<файл lk2nd>`, `<файл bootfs>` и `<файл rootfs>` на актуальные названия файлов из релиза.

## Доступ к системе
Доступ к консоли можно получить через serial по USB с скоростью 115200. Логин: `root`, пароль: `1234`.

## Известные проблемы
- Не работают SIM-карты.
- В PHOSH невозможно отключить автоповорот экрана.
